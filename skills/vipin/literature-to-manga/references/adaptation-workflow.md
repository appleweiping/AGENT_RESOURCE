# Adaptation Workflow

Use this reference when selecting a source, planning the manga format, or deciding how faithful the adaptation should be.

## Candidate Scoring

Score each candidate from 1 to 5 on:

- Narrative clarity: central conflict, stakes, and plot direction are easy to extract.
- Character strength: recurring cast has clear goals, flaws, relationships, and change.
- Scene density: the text contains visual action, locations, confrontations, rituals, travel, or social scenes.
- Visual potential: motifs, costumes, landscapes, props, and repeated symbols can become panels.
- Length fit: the work can fit the requested chapter count without becoming pure summary.
- Adaptation difficulty: lower difficulty means fewer abstract essays, digressions, dense exposition, or unclear chronology.

Recommended weighted score:

```text
total = narrative_clarity * 1.2
      + character_strength * 1.2
      + scene_density * 1.1
      + visual_potential * 1.1
      + length_fit * 0.8
      + (6 - adaptation_difficulty) * 0.8
```

Rank by total score, then explain the top 3 in plain language.

## Adaptation Modes

- Faithful compression: preserve plot order, reduce exposition, merge minor scenes.
- Episodic reinterpretation: keep the premise and cast, reshape chapters around manga cliffhangers.
- Character-first adaptation: focus on one or two protagonists and turn background material into conflict.
- Anthology adaptation: use source episodes as independent chapters with shared motifs.
- Inspired-by adaptation: borrow themes and dramatic situations while clearly marking invention.

Choose one mode explicitly in the adaptation brief.

## Phase Checklist

1. Source intake: identify title, author if known, file path, format, encoding, and extraction quality.
2. Shortlist: rank candidates and record why each is or is not manga-friendly.
3. Beat map: extract major events, turning points, reversals, reveals, and emotional beats.
4. Character map: list goals, conflicts, designs, speaking style, and arc status.
5. Format plan: choose target audience, chapter count, page range, panel density, and cliffhanger cadence.
6. Manga bible: lock recurring visual rules before writing panels.
7. Script: write page and panel units, not prose summary.
8. Prompt pack: convert only visually important panels into image prompts.
9. QA: check continuity, source fidelity, pacing, and prompt consistency.

## Fidelity Rules

- Do not silently turn uncertain source details into facts.
- Mark invented characters, scenes, merged roles, and reordered events.
- Keep direct source quotation short. Prefer paraphrase and adaptation notes.
- Preserve culturally specific terms when they matter, then add concise notes.
- When adapting nonfiction, separate documented events from dramatized scenes.

## Deliverable Order

Produce deliverables in this order unless the user asks otherwise:

1. Candidate dossier
2. Adaptation brief
3. Manga bible
4. Chapter outline
5. Chapter script
6. Storyboard prompt pack
7. QA notes
