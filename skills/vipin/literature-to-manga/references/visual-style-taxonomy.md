# Visual Style Taxonomy

Use this reference when choosing art direction or writing image prompts.

## Style Axes

Define style using these axes instead of vague labels:

- Line: clean, rough, brushy, ink-heavy, delicate, angular, soft.
- Tone: high contrast, gray wash, screentone, watercolor wash, flat shadows, cinematic lighting.
- Character proportion: realistic, semi-realistic, shojo elegance, shonen energy, chibi inserts, gekiga realism.
- Panel mood: quiet literary, suspense, historical epic, street realism, romantic melancholy, mythic fantasy, political drama.
- Camera: intimate close-ups, theatrical staging, documentary framing, action choreography, symbolic compositions.
- Era detail: clothing, architecture, signage, vehicles, weapons, tools, interior design.

## Prompt Components

For each key visual prompt, include:

1. Subject and action
2. Character identity anchors
3. Setting and era details
4. Composition and camera
5. Lighting and tone
6. Manga style axes
7. Continuity constraints
8. Negative constraints

## Character Sheet Prompt Pattern

```text
Full-body character design sheet of {character}, {age}, {role}, wearing {costume},
front view and three-quarter view, key props: {props}, expression set: {expressions},
{line style}, {tone}, {era}, white background, consistent manga model sheet.
Negative: inconsistent face, extra fingers, unreadable text, modern items, wrong costume.
```

## Panel Prompt Pattern

```text
Manga panel, {shot type}, {character} {action} in {setting}, {era details},
composition: {composition}, lighting: {lighting}, mood: {mood},
style: {line/tone/proportion}, continuity: {costume/props/injuries}.
Negative: photorealism, western comic coloring, extra limbs, malformed hands, garbled text.
```

## Continuity Anchors

Maintain a compact continuity list:

- Character face shape, hair, age, height, posture
- Signature clothing and color/value palette
- Props and weapons
- Injuries, scars, glasses, jewelry
- Location layout
- Time of day and weather
- Visual motifs tied to theme

## Style Selection Guidance

- Literary realism: quiet compositions, detailed interiors, restrained expressions, gray wash.
- Historical drama: researched clothing, architecture, crowd staging, ink-heavy texture.
- Martial or action story: dynamic camera, readable choreography, strong silhouettes.
- Psychological story: close-ups, negative space, symbolic inserts, distorted framing.
- Biography: documentary staging, recurring visual motifs, time-period accuracy.
